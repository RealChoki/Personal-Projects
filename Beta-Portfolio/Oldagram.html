Oldagramweb app

HTML & CSS

GitHub Repo Live version

This Web application is the main layout of an Instagram clone. It utilizes the power of vanilla JavaScript along with some well-crafted HTML and CSS skills to create an intuitive and user-friendly interface.

When a user taps on the heart icon, the system already triggers an interactive animation that visually represents the action of liking the post. This animation includes the heart changing color to indicate that the like has been registered. Additionally, the number of likes associated with the post is incremented by one, displaying the updated count to reflect the user's engagement.

The implementation of the like function utilizes JavaScript to handle the logic behind it. Event listeners are set up to detect when a user clicks on the heart icon, triggering a predefined function that updates the like count and applies the necessary visual changes.
